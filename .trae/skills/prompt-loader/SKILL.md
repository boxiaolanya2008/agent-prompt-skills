# 提示词加载器

## 功能描述

本技能专门负责提示词的加载操作，确保在加载过程中严格遵循系统规范，避免格式错误、内容遗漏或解析异常。加载完成后，验证提示词的完整性、准确性和可用性，确保所有功能模块和指令逻辑能够正常运行。

## 核心功能

### 1. 提示词加载
- 读取并解析提示词文件
- 检查文件编码和格式兼容性
- 验证基本语法结构
- 确保无格式错误和解析异常

### 2. 完整性验证
- 检查提示词内容完整性
- 验证功能模块完整性
- 确认指令逻辑完整性
- 检测内容遗漏问题

### 3. 准确性检查
- 验证提示词准确性
- 检查功能模块准确性
- 确认指令逻辑准确性
- 确保无逻辑错误

### 4. 可用性测试
- 测试功能模块可用性
- 验证指令逻辑可用性
- 确保系统正常运行
- 检查性能表现

## 调用时机

在以下情况下调用本技能：
- 用户需要加载提示词文件
- 需要验证提示词格式和内容
- 检查提示词完整性和准确性
- 确保功能模块正常运行
- 进行系统可用性测试

## 操作流程

1. **文件检查**：检查提示词文件是否存在、可读
2. **格式验证**：验证文件格式是否符合规范
3. **内容解析**：解析提示词内容，检查语法结构
4. **完整性检查**：验证所有必要模块和指令是否存在
5. **准确性验证**：检查逻辑准确性和功能正确性
6. **可用性测试**：测试功能模块和指令逻辑的运行状态
7. **结果报告**：生成加载和验证报告

## 错误处理

- 检测并报告格式错误
- 识别内容遗漏问题
- 标记解析异常
- 提供修复建议

## 输出格式

技能执行完成后将提供：
- 加载状态报告
- 验证结果摘要
- 错误和警告列表
- 可用性测试结果
